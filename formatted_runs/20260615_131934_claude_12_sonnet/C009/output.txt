This chunk defines how loan application records are retrieved from storage,
including standard lookups by status or applicant, and specialized queries for
finding stale pending applications, counting high-value approved loans, and

persistence layer's contract for all loan application data access used
elsewhere in the system.

can be counted.
An applicant's currently active pending applications can be retrieved.
