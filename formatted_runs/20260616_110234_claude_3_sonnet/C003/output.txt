business_functional_name: |
Loan Application Submission And Retrieval Endpoints

core_business_functionality: |
This component exposes the HTTP entry points through which borrowers and
officers submit new loan applications, retrieve the details of individual
applications, retrieve all applications associated with a specific borrower,
and update the status of an application as it progresses through the
origination lifecycle. It is responsible for accepting validated inbound
requests and delegating all business processing to the service layer.

business_function: |
Accept a new loan application submission from a borrower or originating
officer.
Return the newly created application record with its assigned reference upon
successful submission.
Retrieve the full details of a specific loan application by its unique
reference number.
Retrieve the list of all loan applications associated with a given borrower,
with pagination support.
Accept a request to update the status of an existing loan application.
Return the updated application record following a successful status change.
Log incoming request details for traceability.
Log the outcome of completed operations for audit support.
